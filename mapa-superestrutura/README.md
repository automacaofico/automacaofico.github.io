# MAPA SUPERESTRUTURA

Dashboard independente FICO para visualizar avanço linear diretamente no traçado ferroviário.

## Fontes

- Planilha online: `automacaofico/super`.
- Geometria: `Geo - KMZ Pacotes.zip`.
- Cobertura: pacotes 01 até 08.

## Funcionamento

O painel busca automaticamente `Modelo Acompanhamento Super - FINAL.xlsx`.

Atualizações feitas pelo portal existente aparecem também neste mapa.

Arquivos Excel locais podem ser carregados para conferência temporária.
